Installation Instructions

  1. Download the extension using the button above
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" in the top right
  5. Click "Load unpacked" and select the extracted folder
  6. The extension will appear in your browser toolbar

How to Use

  1. Click the ScriptSor extension icon in your browser toolbar
  2. Drag and drop your video file or click to select
  3. Wait for the conversion to complete (shows progress)
  4. Click "Upload to ScriptSor" to continue with transcription
  5. Edit your text and export the final video
